# Galleria Frontend

Frontend Angular do projeto Galleria.

## Requisitos

- Node.js 24
- npm

## Instalação

```bash
npm install
```

## Rodar localmente

```bash
npm start
```

Depois acesse o endereço mostrado no terminal.

## Build de produção

```bash
npx ng build --output-hashing=all --configuration production
```

O resultado do build fica na pasta `dist/`.
